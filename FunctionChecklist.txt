FUNCTION CHECKLIST
1. Did you start your function definition with def?
2. Does your function name have only characters and underscore_characters?
3. Did you put an open parenthesis right after the function name?
4. Did you put your arguments after the parenthesis separate by commas?
5. Did you make each argument unique?
6. Did you put a close parenthesis after the arguments?
7. Did you indent by two spaces all lines of code you want in the function?
8. Did you end your function with end lined up with def above?

USE CALL FUNCTION
1. Did you call/use/run this function by typing its name?
Did you put the ( character after the name to run it?
Did you put the values you want into the parenthesis separated by commas?
4. Did you end the function call with a ) character?
5. Functions that don't have parameters do not need the () after them, but would it be clearer if you wrote them anyway?
